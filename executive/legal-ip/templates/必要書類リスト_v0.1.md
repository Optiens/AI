## 弁護士・税理士確認が必要な項目（DoR）

業務フロー定義 v1.4 の DoR（Definition of Ready）に対応：

### 弁護士確認待ち
- [ ] 業務委託契約書（要件定義）テンプレ正式版
- [ ] 業務委託契約書（導入支援）テンプレ正式版
- [ ] 業務委託契約書（保守）テンプレ正式版
- [ ] NDA テンプレ正式版
- [ ] 検収書テンプレ正式版
- [ ] 不成約時の成果物（要件定義書）の著作権帰属確定
- [ ] 損害賠償上限の妥当性（契約金額相当額）

### 税理士確認待ち
- [ ] 見積書テンプレ正式版
- [ ] 請求書テンプレ正式版（適格請求書要件充足確認）
- [ ] 領収書テンプレ正式版（適格請求書要件充足確認）
- [ ] 充当時の消費税処理（成約時充当の会計処理）
- [ ] 印紙税の扱い（電子契約・電子請求書なら不要）
- [ ] インボイス制度対応（適格請求書発行事業者登録番号取得後の更新）

### サイト掲載必須項目
- [ ] 特商法表記（B2B限定の旨）
- [ ] プライバシーポリシー（既存：/privacy-policy）
- [ ] 全自動メールに配信解除リンク付与（特定電子メール法対応）
- [ ] フォーム DoS 対策（既存：reCAPTCHA・レート制限）
