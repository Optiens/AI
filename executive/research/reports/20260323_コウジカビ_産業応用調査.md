## 1. コウジカビ（麹菌）基本情報

### 1.1 分類と特性

| 項目 | 内容 |
|---|---|
| 学名 | *Aspergillus oryzae*（ニホンコウジカビ / 黄麹菌） |
| 分類 | 真菌門 > 子嚢菌綱 > ユーロチウム目 > マユハキタケ科 > コウジカビ属 |
| 近縁種 | *A. flavus*（アフラトキシン産生菌）と遺伝的に極めて近いが、*A. oryzae* はアフラトキシンを産生しない |
| 全ゲノム解読 | 2005年、日本の産学官グループにより完了（約12,000遺伝子） |
| 特徴 | 強力なタンパク質分泌能力（アミラーゼ、プロテアーゼ、リパーゼ等を菌体外に大量分泌） |

出典: [Aspergillus oryzae - Wikipedia](https://en.wikipedia.org/wiki/Aspergillus_oryzae), [PMC - The ancient koji mold as a modern biotechnological tool](https://pmc.ncbi.nlm.nih.gov/articles/PMC10992763/)

### 1.2 伝統的な用途

コウジカビは1,000年以上にわたり日本の醸造・食品産業の根幹を支えてきた。

| 食品 | 麹の役割 |
|---|---|
| 日本酒 | 米のデンプンを糖化 → 酵母がアルコール発酵（並行複発酵） |
| 味噌 | 大豆タンパク質を分解 → うま味成分（アミノ酸）生成 |
| 醤油 | 小麦デンプン + 大豆タンパクの分解 → 複雑な風味形成 |
| 焼酎・泡盛 | 黒麹菌（*A. luchuensis*）によるクエン酸生成 + 糖化 |
| 酢 | 米の糖化 → アルコール発酵 → 酢酸発酵の前段階 |
| 本みりん | 糖化によるブドウ糖・オリゴ糖生成 |
| 甘酒 | 米デンプンの糖化（アルコール発酵なし） |
| 鰹節 | カビ付け工程で水分除去・脂肪分解 |

### 1.3 「国菌」認定

- **2006年10月12日**: 日本醸造学会大会にて「国菌（こっきん）」に認定
- 認定理由:
  1. **食文化への貢献**: 古来より醸造・食品製造に不可欠な微生物
  2. **産業・医薬品への応用**: 高峰譲吉博士が110余年前にタカジアスターゼを抽出（世界初の産業用酵素）
  3. **科学的価値**: 2005年のゲノム解読により、今後さらなる産業応用が期待される
- 発案者: 一島英治博士（「日本からの麹菌の科学技術と文化の発信は、21世紀の世界に大きなインパクトを与える」）

出典: [日本醸造学会 国菌認定](https://www.jozo.or.jp/gakkai/notices/%E3%80%8C%E5%9B%BD%E8%8F%8C%E3%80%8D%E8%AA%8D%E5%AE%9A%E3%80%80%E4%B8%80%E9%83%A8%E6%94%B9%E6%AD%A3%EF%BC%88%E8%8F%8C%E5%90%8D%E5%A4%89%E6%9B%B4%EF%BC%89/), [マルカワみそ](https://marukawamiso.com/spec/aspergillus-oryzae.html)

## 2. 「将来重要になる」理由 -- 最新トレンド

### 2.1 代替タンパク質（Alternative Protein）

菌糸体発酵（mycelium fermentation）による食品タンパク質生産が急成長中。*A. oryzae* の菌糸体は高タンパク質で、肉のような繊維構造を自然に形成する。

- **2025年 Nature論文**: 窒素源を補充した固体麹発酵により、米・トウモロコシのタンパク質含量をそれぞれ2.32倍・1.58倍に増加（12.3%, 13.2%達成）
- **Edible mycelium**: *A. oryzae* の菌糸体を合成生物学ツールキットで改変し、栄養価と食感を向上させる研究（Nature Communications, 2024）
- **マイコプロテイン企業**: 菌糸体ベースの代替肉がVC投資の注目分野（2024年の発酵技術への投資額は6.51億ドル、前年比+43%）

出典: [Nature - Fast and easy edible protein production by nitrogen-supplemented koji fermentation](https://www.nature.com/articles/s41538-025-00463-2), [Nature Communications - Edible mycelium bioengineered](https://www.nature.com/articles/s41467-024-46314-8)

### 2.2 精密発酵（Precision Fermentation）

遺伝子改変した微生物にバイオリアクターで特定タンパク質を生産させる技術。*A. oryzae* はFDA GRASの長い実績と優れた分泌能力により、精密発酵の「宿主」として最有力候補の一つ。

- **ウシβ-ラクトグロブリン生産**: 工業用 *A. oryzae* 株で乳タンパク質を大量生産する研究（2025年、SAGE Journals）
- **課題**: 目標コスト 10 USD/kg（2025年） → 1 USD/kg（2035年）への低減。現在の世界のバイオリアクター容量6,100万Lに対し、2030年までに100億Lが必要（約160倍のスケールアップ）

出典: [SAGE Journals - Enhanced Production of Bovine beta-Lactoglobulin in A. oryzae](https://journals.sagepub.com/doi/10.1177/15509087251398318), [三井物産戦略研究所レポート](https://www.mitsui.com/mgssi/ja/report/detail/__icsFiles/afieldfile/2024/02/15/2402t_sawano.pdf)

### 2.3 フードテック -- 発酵食品イノベーション

- **ORYZAE（オリゼ）**: 米麹発酵から砂糖代替甘味料を開発。2022年に売上16倍成長、米麹グラノーラ累計7万食突破
- **麹の英語圏認知拡大**: 「koji」は欧米のクラフト食品・レストラン業界で急速に認知度上昇中。Noma（デンマーク）やRene Redzepi等が麹発酵技術を世界に発信

### 2.4 サステナビリティ -- 食品廃棄物のアップサイクル

- **NoMy（ノルウェー）**: 食品廃棄物を菌糸体発酵でマイコプロテインに変換するAI駆動型サービス。2025年にEUR 125万を調達（日本甜菜製糖がリード投資家）
- **Kynda（ドイツ）**: 食品廃棄物から菌糸体肉を製造。EUR 300万のシード資金を調達（2025年Q2に工場稼働予定）

出典: [EU-Startups - NoMy raises EUR1.25M](https://www.eu-startups.com/2025/05/power-of-fungi-norwegian-startup-nomy-raises-e1-25-million-to-scale-its-sustainable-protein-alternative/), [Green Queen - Kynda EUR3M funding](https://www.greenqueen.com.hk/kynda-mycelium-meat-mycoprotein-funding-protein/)

### 2.5 バイオマテリアル

菌糸体は食品以外にも幅広い素材応用が進行中。

| 分野 | 応用例 | 状況 |
|---|---|---|
| レザー | MycoWorks（菌糸体レザー「Reishi」）| 商用化済み、Hermes等と提携 |
| パッケージング | Ecovative Design（菌糸体梱包材）| 商用化済み、発泡スチロール代替 |
| 建材 | 断熱パネル・音響パネル | 研究段階～パイロット |
| テキスタイル | 菌糸体繊維のファッション素材 | 研究段階～パイロット |

- *A. oryzae* 自体よりも *Ganoderma*（霊芝）や *Pleurotus*（ヒラタケ）が多く使われるが、NaOH脱アセチル化によりキチンをキトサンに変換し、機械的特性を向上させる技術は属を問わず応用可能

出典: [Springer - Mycelium-Based Breakthroughs](https://link.springer.com/article/10.1007/s43615-025-00539-x), [ACS - Improving Tensile Properties of Aspergillus Mycelium](https://pubs.acs.org/doi/10.1021/acssuschemeng.5c07392)

### 2.6 酵素生産プラットフォーム

*A. oryzae* は世界の産業用酵素市場の約30%を占める主要生産菌。

| 酵素 | 用途 | 市場規模 |
|---|---|---|
| alpha-アミラーゼ | 高果糖コーンシロップ、パン、繊維加工 | 約3.2億ドル（2024年、グローバル） |
| プロテアーゼ | 食品加工、洗剤、皮革 | 産業酵素市場全体の約30% |
| リパーゼ | 油脂加工、洗剤 | 成長中 |
| セルラーゼ | バイオ燃料、繊維 | 成長中 |

- 産業酵素市場全体: 約80億ドル（2025年推定）、CAGR 5.5%以上で成長中
- *A. oryzae* のゲノムには135の分泌型プロテアーゼ遺伝子が予測されている

出典: [PMC - Aspergillus oryzae as a Cell Factory](https://pmc.ncbi.nlm.nih.gov/articles/PMC11051239/), [Arizton - Industrial Enzymes Market](https://www.arizton.com/market-reports/industrial-enzymes-market-size-analysis)

### 2.7 ユネスコ無形文化遺産「伝統的酒造り」登録の影響

- **2024年12月4日**: ユネスコ政府間委員会が「伝統的酒造り」の無形文化遺産登録を全会一致で決定
- **定義**: 杜氏・蔵人等がこうじ菌を用い、長年の経験に基づき築き上げた酒造り技術（500年以上の歴史）
- **輸出への影響**: 日本酒の輸出額は2013年の約100億円 → 2023年に約400億円（4倍増）。2013年の「和食」登録が輸出拡大の一因であり、今回の登録でさらなる拡大が期待される
- **麹の世界的認知向上**: 酒造りの核心が「麹菌と酵母の共同作用」であることが国際的に認知され、麹関連のビジネス・研究への追い風

出典: [nippon.com - ユネスコ無形文化遺産になった日本の伝統的酒造り](https://www.nippon.com/ja/japan-topics/g02471/), [国税庁 - ユネスコ無形文化遺産「伝統的酒造り」](https://www.nta.go.jp/taxes/sake/koujikin/index.htm)

### 3.3 「koji」の英語圏認知度

- 欧米のクラフト食品・発酵食品コミュニティでは「koji」はすでに一般的な用語
- Noma（世界一に選ばれたデンマークのレストラン）のRene Redzepi氏が麹発酵の書籍を出版、世界的に認知を加速
- Amazon.com で "koji" 検索すると、麹スターター、麹関連書籍、調味料が多数ヒット
- 代替タンパク質文脈で "koji mycelium" は投資家・メディアに頻出するキーワード

### 3.4 Meati Foodsの教訓

Meati Foodsは累計5億ドル以上を調達しながら、2025年5月に400万ドルでの売却報道が出た。原因は銀行のテクニカルデフォルトにより現金準備の2/3を喪失したこと。代替タンパク質市場では「巨額調達 = 成功」ではなく、ユニットエコノミクスの健全性が重要であることを示す事例。Prime Rootsの「ユニットプロフィタブル + 慎重成長」戦略との対比が顕著。

## 4. 培養条件と環境制御技術との共通性

### 4.1 コウジカビの培養条件

| パラメータ | 最適範囲 | 許容範囲 | 備考 |
|---|---|---|---|
| **温度** | 32～36 C | 成長限界: 44 C | 酵素活性最適: 35～40 C |
| **湿度** | 80%以上 | 水分活性 0.8以上 | 固体培養では基質水分が重要 |
| **pH** | 5.0～6.0 | 発芽可能: pH 2.0～8.0 | 酵素生産最適: pH 5 |
| **酸素** | 好気性（酸素豊富な環境） | -- | 固体培養では気中菌糸が酸素取込を担う。タンパク質発現は酸素依存 |
| **CO2** | 低濃度 | -- | 高CO2は成長抑制 |

出典: [Kenyon College - Aspergillus oryzae microbewiki](https://microbewiki.kenyon.edu/index.php/Aspergillus_oryzae), [ScienceDirect - Aspergillus oryzae overview](https://www.sciencedirect.com/topics/agricultural-and-biological-sciences/aspergillus-oryzae)

### 4.2 固体培養（SSF）vs 液体培養（SmF）

| 項目 | 固体培養（SSF） | 液体培養（SmF） |
|---|---|---|
| 伝統的用途 | 味噌、醤油、日本酒の製麹 | 産業用酵素・有機酸の大量生産 |
| タンパク質分泌量 | 高い | SSFより少ない |
| スケールアップ | 困難（均一制御が難しい） | 容易（バイオリアクターで制御可能） |
| IoTセンサーの適用 | 温湿度・CO2が特に重要 | 溶存酸素・pH・温度が特に重要 |
| 代替タンパク質文脈 | 麹ベースの食品（Prime Roots等） | 精密発酵（Formo等） |

### 4.3 水耕栽培の環境制御技術との共通性

**植物工場とコウジカビ培養で共通するセンサー・制御項目**:

| 共通制御項目 | 植物工場 | コウジカビ培養 |
|---|---|---|
| 温度 | 20～28 C（作物依存） | 32～36 C |
| 湿度 | 60～80% | 80%以上 |
| CO2濃度 | 800～1200ppm（光合成促進）| 低濃度管理（過剰CO2抑制）|
| pH | 養液pH 5.5～6.5 | 培地pH 5.0～6.0 |
| 溶存酸素 | 養液中のDO管理 | 液体培養でのDO管理 |
| 照明/光 | LED光量・光質制御 | 不要（暗条件が一般的） |

共通点: 温湿度・pH・CO2のリアルタイム監視と制御が核心技術であり、Optiensの「汎用IoT環境モニタリングSaaS」のセンサーインテグレーション・ダッシュボード・アラート機能はそのまま応用可能。

### 4.4 バイオリアクターの種類と価格帯

| スケール | 容量 | 価格帯（USD） | 用途 |
|---|---|---|---|

| パイロットスケール | 10～1,000 L | 50,000～200,000 | プロセス開発・スケールアップ試験 |

- 中国製の廉価モデルは10,000ドル～で入手可能
- 中古バイオリアクターは10,000～100,000ドルが一般的価格帯

## 6. 規制・認証

### 6.1 食品安全性

| 機関 | 評価 |
|---|---|
| **FDA（米国）** | GRAS（Generally Recognized as Safe）認定。多数のGRAS通知が異議なく受理済み（GRN 8, 34, 43, 75, 90, 103, 106, 113, 122, 142, 201, 510, 811, 829, 979, 982, 1039等） |
| **JECFA（FAO/WHO合同）** | 2020年に安全な食品酵素源として結論。有害・毒性影響の記録なし |
| **WHO** | 安全性を支持 |

- *A. oryzae* は *A. flavus*（アフラトキシン産生菌）と遺伝的に近いが、アフラトキシンやその他の発がん性代謝物の産生記録はない
- 2025年にはマイコトキシンフリーの *A. oryzae* 株系統が工業スケールの代替タンパク質生産向けに確立されたことが報告（PMC）

出典: [FDA GRAS Notice GRN 829](https://www.fda.gov/media/132948/download), [PMC - Mycotoxin-free A. oryzae strain lineage](https://pmc.ncbi.nlm.nih.gov/articles/PMC11993488/)

### 6.2 各国・地域の規制状況

| 地域 | 規制枠組み | *A. oryzae* の位置づけ |
|---|---|---|
| **米国** | FDA GRAS / FSIS（肉代替品はUSDA管轄） | GRASとして広く認定。酵素・発酵食品に長い使用歴 |
| **EU** | Novel Food Regulation (EU) 2015/2283 | 伝統的使用歴がある酵素・発酵食品は既存食品扱い。新規タンパク質（菌糸体肉等）はNovel Food申請が必要な場合あり |
| **日本** | 食品衛生法・食品表示法 | 伝統的に使用されており特段の規制なし。遺伝子組換え *A. oryzae* 使用の場合はカルタヘナ法の対象 |

### 6.3 規制上の優位性

*A. oryzae* が精密発酵・代替タンパク質の宿主として注目される最大の理由の一つが規制面の優位性:

- **1,000年以上の安全な使用歴**: 他の微生物プラットフォームにはない圧倒的な実績
- **複数のGRAS通知が異議なし**: FDA承認プロセスが円滑
- **WHO/JECFA評価済み**: 国際的な安全性コンセンサス
- **マイコトキシン非産生の確認**: 近縁種 *A. flavus* との明確な区別が確立

## 7. Optiensへの示唆

### 7.1 AquaLoop Platformの横展開可能性

コウジカビ培養（固体・液体ともに）で必要な環境モニタリングは、Optiensが植物工場向けに開発するセンサーインテグレーション・ダッシュボード・アラート機能と高い親和性を持つ。

| Optiens機能 | 植物工場での用途 | 麹/発酵での用途 |
|---|---|---|
| 温湿度モニタリング | 栽培環境の最適化 | 製麹室・発酵室の管理 |
| pHモニタリング | 養液管理 | 培地・もろみの管理 |
| CO2モニタリング | 光合成促進 | 好気発酵の管理 |
| アラート | 環境逸脱の即時通知 | 発酵異常の即時通知 |
| AI異常検知 | 生育不良の早期発見 | 雑菌汚染・発酵異常の検出 |
| データ可視化 | 収穫予測・環境トレンド | 発酵進行の可視化・品質管理 |

### 7.2 ロードマップとの整合性

CLAUDE.mdのロードマップにおいて、Phase 3（2028/4～2029/3）で「醸造・発酵への横展開」が明記されている。本調査結果は、その横展開先としてコウジカビ/麹関連産業が有望であることを裏付ける。

### 7.3 ターゲット顧客候補

- 日本酒蔵（IoT導入による製麹・発酵管理の近代化ニーズ）
- 味噌・醤油メーカー（品質安定化・データ蓄積ニーズ）
- クラフト発酵食品メーカー（小規模ゆえに専任技術者不在）
- 精密発酵スタートアップ（バイオリアクターの環境モニタリングニーズ）

### 7.4 注意点

- 本レポートはコウジカビの市場機会を整理したものであり、Optiensのターゲット変更を提案するものではない
- ロードマップの Phase 0～2 は植物工場に集中すべきであり、醸造・発酵は Phase 3 以降の計画に従う
- 代替タンパク質市場はMeati Foodsの事例が示す通り、過剰投資リスクがあることに留意
