# サービスチケット定義（2026-05-17）

## 結論

OptiensのAI支援事業では、低単価の人間稼働を固定化しないため、詳細版AI活用診断とスポット相談チケットを明確に分ける。

- 詳細版AI活用診断: 5,500円（税込）。MTGなし。AI診断官βを入口・体験デモ・追加ヒアリング・診断メモ生成に使い、レポート価値を補強する
- 単発AI相談: スポット相談チケット1枚。60分目安の人間相談。導入支援費へ充当しない
- 有償要件定義: スポット相談チケット2枚。導入支援前の有料スコーピング。導入支援成約時は2枚分を初期費用へ充当する
- 簡易実装・軽微な自動化: 原則3枚から
- 4枚以上、外部API連携、本番システム構築、認証、個人情報、継続運用を含むものは個別見積へ切り替える

## スポット相談チケット

| 項目 | 内容 |
|---|---|
| 価格 | 1枚 ¥33,000（税込、税抜¥30,000） |
| 購入単位 | 1〜3枚は購入申込フォームで受付。4枚以上は個別確認 |
| 購入申込 | `/spot-ticket-purchase` |
| 通常利用申請 | `/spot-ticket-redeem` |
| 説明LP | `/spot-ticket` |
| 使途 | 単発AI相談、有償要件定義、簡易実装・軽微な自動化 |

スポット相談チケットは、相談・要件整理・小規模作業の支払い単位として扱う。購入申込後、顧客へ口座振込案内・請求情報・振込完了通知URLを自動送信し、freeeの入金照合後にチケット番号を発行する。通常の自動照合は毎朝9時のCronで行い、顧客は振込完了通知URLから任意のタイミングで即時照合を試せる。急ぎの場合は、お問い合わせフォームで申込番号・振込人名義・振込日時を受け取り、手動確認の入口とする。

## AI診断官β

AI診断官βは、スポット相談チケット1枚で販売する独立商品ではない。入口、体験デモ、情報収集ソース、詳細版AI活用診断の価値増強として扱う。

- `/ai-diagnosis-officer` で購入前に公開デモを提供する
- 音声モードとテキストモードを併設する
- 音声モードはβ版として、静かな環境での利用を推奨する
- 声の選択機能は提供しない。男性/女性音声への切り替えをできたように装わない
- 会話後の成果物は文字起こしではなく診断メモとする
- `response.done` のusageをセッション単位で集計し、推定API原価を管理画面に出す
- 正式見積、契約条件、法務・税務・医療判断は確定しない

詳細版レポート後に追加情報が必要な場合、AI診断官βで聞き取り、診断メモとしてAI化候補、AI化しない方がよい範囲、制約条件、必要データ、次アクションへ構造化する。人間による相談が必要な場合は、単発AI相談へ切り替える。

## 単発AI相談

単発AI相談は、スポット相談チケット1枚で実施する60分目安の人間相談である。

対象は、詳細レポートやAI診断官βの診断メモを見たうえで、短時間で方向性を確認したい顧客とする。成果物は簡易議事メモまたは次アクション整理までに限定し、実装、要件定義書、正式見積、調査代行を含めない。

導入支援費には充当しない。代表の時間単価を低く固定しないため、月間受付数に上限を設け、複雑化した場合は有償要件定義または個別見積へ切り替える。

## 有償要件定義

有償要件定義は、導入支援に進む前の有料スコーピングとして、スポット相談チケット2枚で実施する。キックオフMTGではない。キックオフMTGは導入支援契約後に、実装開始に向けて仕様・進め方を最終確認する場とする。

対象は、導入支援を直接希望する顧客、または実装範囲・費用・優先順位が曖昧なまま見積を出すと危険な顧客とする。対象業務、実装範囲、前提条件、含む範囲・含まない範囲、概算費用、次アクションを整理する。

導入支援の本契約に進む場合は、支払済みのスポット相談チケット2枚分を本契約の初期費用へ充当する。本契約に進まない場合も、要件整理の対価として返金しない。

## 簡易実装・軽微な自動化

簡易実装・軽微な自動化は、原則としてスポット相談チケット3枚から受ける。

範囲は、1業務・1フロー、既存ツール・既存データ前提、設定または簡易実装、動作確認1回、軽微な修正1回までに限定する。本番運用保証、継続監視、複雑な例外処理、外部API連携、認証、個人情報、決済を含む場合は個別見積へ切り替える。
