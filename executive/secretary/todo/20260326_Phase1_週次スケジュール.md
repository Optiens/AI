# Phase 1 週次スケジュール（2026年4月〜9月）

## 前提条件
- SF業務委託: 月・火・木・金（実稼働0.5人月）、AI活用で効率化済み
- **水・土・日**: Optiens専用日（週3日フル + 平日夜）
- 物理作業（配管・配線・栽培）はAIに委任不可 → 水・土・日に集中
- 事務・調査・申請書はAI（Claude Code）に80%委任可能

## 月別フォーカス

| 月 | メインテーマ | サブテーマ |
|---|---|---|
| **4月** | 法人設立 + IoTシステム構築 | 栽培はまだやらない |
| **5月** | IoT完成 + 初回播種 | 補助金申請提出 |
| **6月** | 栽培データ蓄積 | 廃校の行政回答待ち |
| **7月** | 反復栽培 + 品質安定化 | レストランリサーチ |
| **8月** | サンプル提供開始 | 販路テスト |
| **9月** | Phase 2移行判断 | 廃校確保の最終判断 |

## Week 13-16（7月: 反復栽培 + レストランリサーチ）

**方針: 2-3サイクル目を回してデータの再現性を確認。並行してレストラン情報収集。**

### Week 13-14（7/1-7/13）

### Week 15-16（7/14-7/27）
- サイクル3播種
- **レストランリストアップ（E-21）**: 北杜市周辺のイタリアン・フレンチ・カフェ5-10軒をWeb調査
- 各店のメニュー・ハーブ使用状況を確認
- 道の駅こぶちさわに出荷条件を電話確認

**Week 16完了条件**: 3サイクル分のデータ、レストラン候補リスト完成

## Week 17-20（8月: サンプル提供開始）

**方針: 品質が安定したことを確認してからサンプル提供。焦って持っていかない。**

### Week 17-18（7/28-8/10）
- サイクル3収穫 → CV（変動係数）の暫定算出
- CVが30%以上なら原因分析・パラメータ修正
- CVが20%以内なら品質安定と判断

### Week 19-20（8/11-8/24）
- **品質安定を確認した場合のみ**: レストラン2-3軒に無料サンプル提供（E-22）
- サンプル提供時に栽培データダッシュボードを見せる（差別化）
- フィードバック収集（量・品質・頻度のニーズ）

**Week 20完了条件**: サンプル提供済み（品質安定の場合）、4サイクル以上のデータ蓄積

## Week 21-24（9月: Phase 2移行判断）

### Week 21-22（8/25-9/7）
- サイクル5のデータ収集
- Phase 2移行条件の暫定チェック:
  - [ ] 1株コスト ≦ 販売価格の40%
  - [ ] 週次収穫量CV ≦ 20%（直近3サイクルで判定）
  - [ ] レストラン2軒以上からテスト発注の意向（継続発注の約束は現実的に困難なため緩和）
  - [ ] IoTシステム1ヶ月連続稼働（計画外停止15分以内）

### Week 23-24（9/8-9/21）
- **Phase 2移行判断会議**（1人だが、Claude Codeで客観的にデータレビュー）
- 4条件のうち未達がある場合:
  - 1-2項目未達 → Phase 1を3ヶ月延長（〜12月）して再チャレンジ
  - 3項目以上未達 → 根本的な見直し（品目変更・方式変更を検討）
- 廃校教室の行政手続き状況の最終確認
- 廃校NGの場合の代替シナリオ発動（コンテナ・ガレージ・空き店舗）

## 週間タイムブロック（標準パターン）

```
月: SF業務 (8h)
火: SF業務 (8h)
水: Optiens事務・開発 (8h) ← AIに委任可能なタスク中心
木: SF業務 (8h)
金: SF業務 (8h)
土: Optiens物理作業 (6-8h) ← 配線・配管・栽培・計測
日: Optiens開発・調査 (4-6h)
```

**週あたり**: SF 32h + Optiens 18-22h = 合計50-54h

## リスク対策

| リスク | 対策 |
|---|---|
| SF業務で急なトラブル | 水・土・日のOptiens時間に影響しない。木金で吸収 |
| IoTのハードウェアトラブル | 予備パーツ確保（DS18B20×5個、リレー×2個）。切り分け手順を事前にドキュメント化 |
| 栽培失敗（根腐れ・カビ等） | 最初の2サイクルは「学習期間」と割り切る。失敗データも記録 |
| 廃校が借りられない | 7月までに行政回答を得る。NGなら代替施設を8月中に確保 |
| 補助金不採択 | Phase 2の規模を縮小（ラック2台→自宅継続）。自己資金で最低限の機器のみ購入 |
