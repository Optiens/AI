# ブログ要約投稿テンプレ（媒体別3種）

2026-05-13 初版。Optiens公式ブログを各SNSで紹介する際の投稿テンプレ。
ハンドル統一：X `@daichi_optiens` / note `daichi_optiens` / Facebook 実名（Daichi Takahashi）。

## 共通ルール

- **トーン**：「AIで楽しいことをやっている会社」。煽り口調（激変・革命・最強・神）はNG。
- **媒体ごとにリライト必須**：同じブログでも3媒体でコピペしない（アルゴリズム対策）。
- **CTAは1つに絞る**：基本はブログ本文URL、または `/free-diagnosis`。
- **AIプロバイダー名**：ブログ要約の文脈では出してOK（Claude / ChatGPT / Gemini）。
- **数値・固有名詞**：ブログ本文と齟齬が出ないように要確認。
- **NG表現**：「ベンダーロックイン」「最先端AI」「業界を激変させる」「神アプデ」。

## ハッシュタグ運用セット

| 用途 | タグ候補 |
|---|---|
| 基本セット（X・note共通） | #AI業務活用 #中小企業DX #AIエージェント |
| 地域フック（任意） | #北杜市 #山梨 #リモートワーク |
| 業界トピック寄り | #ClaudeCode #ChatGPT #生成AI #業務自動化 |
| Facebookは原則ハッシュタグ不要 | （文中に自然に含める程度） |

1投稿あたりタグは3個まで。多用するとリーチが下がる。

## X用テンプレ（3-5ツイートのスレッド）

### 構造

```
ツイート1（フック / 140字以内 / 絵文字1-2個）
　└ 「〜という話」「〜知ってますか？」で引き込む
　└ ブログ本文URLはここでは出さない

ツイート2-4（要点を1ツイートずつ）
　└ 1ツイート1要点。箇条書きでもOK
　└ 具体的な数字・固有名詞を1つは入れる

最終ツイート（CTA）
　└ ブログ全文URL
　└ ハッシュタグ3個
　└ 「続きはブログで」「詳しくはこちら」
```

### 注意点

- 1ツイート目に絵文字を入れすぎない（1-2個まで）。
- 砕けた語尾はOK（「〜ですね」「〜やつ」「〜って話」）。ただし「マジで」「ヤバい」はNG。
- 引用RTで補足を入れる運用は週1〜2回程度に留める。

## note用テンプレ（300-500字）

### 構造

```
リード共感（1-2文 / 70字程度）
　└ 読者の悩み・状況に寄せる

要点3-5個（箇条書き）
　└ 各要点は1行40-60字
　└ 抽象論より具体名・数字優先

締めの誘導文（2-3文）
　└ 「詳細はOptiensブログにまとめました」
　└ ブログ全文URL
　└ Optiensサイトトップ（任意）

ハッシュタグ（noteのタグ機能 / 3個）
```

### 注意点

- noteのタイトルはブログタイトルと変える（重複SEO対策）。
- 本文中に画像を1枚入れると読了率が上がる（ブログのアイキャッチを流用OK）。
- 「リライト」ではなく「誘導」が目的。本文を全部書き写さない。

## Facebook用テンプレ（500-800字）

### 構造

```
個人的なエピソード・気付き（150-200字）
　└ 「先日〜していて」「最近〜していると」で始める
　└ 北杜市の日常・子育て・移住生活ネタは強い（短く1箇所）

事業視点での解釈（200-300字）
　└ そのエピソードから何が言えるか
　└ 中小事業者にとっての意味を1つ加える

ブログ全文への誘導（100-150字）
　└ 「詳しくはブログに書きました」
　└ ブログ全文URL

写真1枚（必須前提）
　└ 北杜市の風景・作業中のPC画面・コーヒーなど
　└ AI画像生成のアイキャッチ流用も可
```

### 注意点

- Facebookは実名・地元コミュニティ前提。商工会・移住者コミュニティを意識した文体。
- ハッシュタグは控えめ。文中に自然に1-2個含む程度。
- 「私は」より「うちは」「Optiensでは」のほうが地元向けの距離感に合う。

## 投稿時刻ガイド

| 媒体 | 推奨時刻 | 補足 |
|---|---|---|
| X | 平日 朝7-9時 / 夜20-22時 | スレッド投稿はまとめて1度に |
| note | 平日 昼12-14時 | 公開後にX・Facebookで告知 |
| Facebook | 平日 朝7-9時 | 写真必須・実名前提 |

土日は投稿しない方針（ビジネス層向けのため）。

## 運用フロー（ブログ1本あたり）

1. ブログ公開（Optiens本サイト）
2. 当日中にX版スレッドを投稿
3. 翌日 noteで誘導記事を投稿
4. 2-3日後 Facebookでエピソード型投稿
5. 1週間後にX再掲（角度を変えて要点1個だけ抜粋）

媒体間で同日投稿しない。リーチを分散させる。
