## 1. 山梨県の独自補助金

### 1-1. 山梨県IoT活用支援事業費補助金 — 適合度B

- **所管**: 山梨県産業労働部 成長産業推進課
- **補助額**: 過去実績では上限100万〜200万円程度（年度により変動）
- **補助率**: 1/2以内
- **公募時期**: 例年4月〜5月頃（2026年度は未発表）
- **適合性**: IoT（RPi×センサー×MQTT）による水耕栽培の自動化・データ可視化は対象になり得るが、自作システムのため対象外リスクあり
- **アクション**: 4月に山梨県成長産業推進課に電話確認（055-223-1534）

### 1-2. 山梨県施設園芸等経営強化支援事業 — 適合度C

- **補助額**: 上限300万円（補助率2/3以内）
- **制約**: 「化石燃料を使用して施設園芸を営む農業者」が主対象。LED水耕栽培は該当しない可能性
- 前回調査と同様にC

---

## 2. 北杜市の独自支援制度

### 2-1. 北杜市チャレンジ農業助成金 — 適合度B

- **所管**: 北杜市農業振興課（0551-42-1350）
- **補助額**: 資材費等の1/2以内（上限最大100万円程度）
- **対象**: 高収益農業へのチャレンジ。種苗・栽培資材・パッケージ資材等
- **制約**: 認定農業者 or 認定新規就農者の要件がある可能性
- **アクション**: 農業振興課に電話で合同会社として副業農業者が申請可能か確認

---

## 3. デジタル化・AI導入補助金2026 — 適合度B

- **公募**: 1次 2026/5/12、2次 2026/6/15
- **クラウド利用料は最大2年分まで補助対象**（ただし登録ITツールに限る）
- **Claude Code Max・Google Oneは対象外**（登録ITツールではないため）
- **freee会計は対象可能**（登録ITツール）
- **アクション**: 登録ITツール一覧で農業向けIoTサービスを検索。2次（6/15）での申請が現実的

---

## 4. 事業再構築補助金 → 不可

- **2025年度で終了**
- 後継「中小企業新事業進出補助金」は**最低1期分の決算書が必須**で設立直後は申請不可

---

## 5. スマート農業関連補助金（農水省）— 適合度C

- 産地・団体単位での申請が基本。個人・小規模では困難
- **みどりの食料システム法認定**（適合度B）は中長期で有効: 機械32%・建物16%特別償却、補助金優先採択
- Phase 2移行時に認定申請を検討

---

## 6. クラウドサービス費用が対象になる補助金

| サービス | デジタル化・AI導入補助金 | 持続化補助金 |
|---|---|---|
| Claude Code Max | **不可**（登録ITツール外） | **不可** |
| Google One | **不可**（登録ITツール外） | **不可** |
| freee会計 | **可能**（登録ITツール） | 不可 |
| 農業IoT SaaS（farmo等） | **可能性あり** | 不可 |

**結論: Claude Code Max・Google One等の汎用サブスクは現行制度では補助対象外**
