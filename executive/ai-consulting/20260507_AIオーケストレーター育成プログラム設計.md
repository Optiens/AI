# AIオーケストレーター育成プログラム — サービス設計（v0.1）

**作成日**: 2026-05-07
**ステータス**: 設計案・CEOレビュー待ち
**対象**: 小規模事業者（1〜30名規模）

---

## 1. プログラムの位置づけ

### Optiensの既存サービス構成
```
[活用診断] → [導入支援] → [保守]
```

### 育成プログラム追加後
```
[活用診断] → [導入支援] → [保守]
                              ↓
                       [AIオーケストレーター育成プログラム]
```

### なぜ必要か
- 全自動化は不可能（エラー・仕様変更・サービス切替への対応必要）
- 保守契約は「最低限の継続性」のみ。「進化させる」役割は別途必要
- 中小事業者には専任職を雇う余裕がないため、**社内1名を兼任で育てる**のが現実解
- 業界では2026年に「AIオーケストレーター」が確立された職種となっている

### サービスのゴール
**「外部コンサルに頼り続けない、自走できる体制づくり」**

---

## 2. ターゲットと典型的な顧客像

### 主要ターゲット
- 1人法人〜30名規模の中小事業者
- AI 導入はある程度進んでいるが、運用に課題を感じている
- 経営者が判断と創造に集中したいが、AI関連の調整に時間を取られている

### 想定ペルソナ

**ペルソナA: 1人法人の経営者（自分が育つ）**
- 5〜15名規模の士業・コンサル・IT業
- 経営者自身がオーケストレーター候補
- 学習コストを最小化したい
- 価格に敏感、月10万円以下が現実的

**ペルソナB: 中小企業の現場エース（社員を育てる）**
- 10〜30名規模の製造業・サービス業
- 業務に精通したエース1名を育てたい
- 経営者が決裁、社員が実務担当
- 投資対効果を重視、ROIが見える形を求める

**ペルソナC: 移行期の中堅事業者**
- 30名前後で組織化フェーズ
- AI 導入を組織レベルで進めたい
- 複数名を育てる必要があるかも
- 中長期投資が可能、月20〜30万円帯

## 3. プログラム階層設計（4ティア）

### Tier 1: セルフ学習プラン（買い切り）

**価格**: ¥39,800（買い切り）

**内容**:
- 録画講義 (8時間相当)
  - AIオーケストレーターの役割と責任
  - 業務見える化の3視点（鳥/虫/人）
  - AIエージェント設計の基礎
  - ハーネス設計の5パターン
  - 監査ログとリスク管理
  - プロンプト設計と継続改善
- 教材PDF（約100ページ）
- 課題ワークシート（10個）
- メールでの質問対応（1ヶ月、5回まで）

**ターゲット**: ペルソナA（経営者本人が学習）
**期間**: 自分のペースで（推奨1〜2ヶ月）
**サービス提供形態**: オンライン完結

**ターゲット**: ペルソナA、ペルソナBの一部

**ターゲット**: ペルソナB、ペルソナCの一部

## 5. プログラム共通カリキュラム（全Tier共通の核）

### モジュール1: AIオーケストレーターとは（1時間）
- 業界での職種定義（SmartRecruiters 2026 / HBR）
- 大企業版 vs 中小事業者版の違い
- 4つの実務スキル
- Optiensの三層構造

### モジュール2: 業務見える化の3視点（2時間）
- 鳥の目（業務フロー俯瞰）
- 虫の目（アクションレベル分解）
- 人の目（イレギュラー対応）
- 演習: 自社業務のマップ作成

### モジュール3: AIに任せる/人間が残す判断（1時間）
- 形式知 / 共通知 / 暗黙知の3層分類
- 100%自動化を目指さない理由
- 属人性の再現性アプローチ

### モジュール4: AIエージェント設計の基礎（2時間）
- システムプロンプト設計
- ツール定義と権限設定
- ワークフロー設計
- 演習: 簡易エージェント設計

### モジュール5: ハーネス設計の5パターン（1時間）
- 最小権限スコープ
- 不可逆操作の人間承認ゲート
- 出力スキーマ検証
- 実行回数・コスト上限
- 構造化監査ログ

### モジュール6: 継続改善とリスク管理（1時間）
- モニタリング指標の設計
- 静かな精度劣化への対応
- セキュリティとコンプライアンス
- 7つのリスクチェックポイント

## 6. 営業・販売戦略

### 主要販路
1. **既存顧客のアップセル**: 導入支援 / 保守契約者向けに案内
2. **無料AI活用診断からの導線**: 診断レポートで「育成が必要」を明示
3. **ブログ記事からの集客**: G群（コンサル実務論）+ H群（オーケストレーター）でインバウンド
4. **無料セミナー**: 月1回のオンラインセミナーで Tier 1〜2 を紹介

### 入口商品（フリー〜ロー価格）
- **無料AI活用診断**: 既存サービス
- **無料セミナー**: 月1回・60分・参加無料
- **¥5,500 詳細AI診断レポート**: 既存サービス

### コンバージョン目標
- 無料診断 → ¥5,500 詳細診断: 10%
- ¥5,500 詳細診断 → Tier 1 セルフ学習: 5%
- Tier 1 → Tier 2 ライト育成: 20%
- Tier 2 → Tier 3 スタンダード育成: 15%

### 月間販売目標（Phase A2 想定）
- Tier 1: 月3件 ≒ ¥120,000
- Tier 2: 月1件 ≒ ¥98,000（初月）
- Tier 3: 半年に1件 ≒ ¥150,000（初月）
- 月間売上目標: ¥250,000〜400,000

## 7. 提供体制と運用

### 講師
- 代表（髙橋大地）が主担当
- 教材は AI エージェントで作成・更新

### 工数試算
- Tier 1: 教材作成のみ（初期投資 80時間）→ 販売後はサポートのみ（月5時間/件）
- Tier 2: メンタリング月2時間/件 + 質問対応月3時間 ≒ 月5時間/件
- Tier 3: メンタリング月4時間/件 + 実装支援月10時間 ≒ 月14時間/件
- Tier 4: メンタリング月8時間/件 + 実装支援月30時間 ≒ 月38時間/件

### 並行受注の上限
代表の稼働を月160時間と仮定:
- Tier 1: 上限なし（自学）
- Tier 2: 同時に最大10件
- Tier 3: 同時に最大4件
- Tier 4: 同時に最大2件

## 9. リスクと対応

| リスク | 影響 | 対応策 |
|---|---|---|
| 大手研修会社が中小事業者向けに参入 | 受注減 | 中小特化・実装連動・少人数制で差別化 |
| 教材の陳腐化が早い（AI業界の変化速度） | 価値低下 | 月次アップデート・現場知識を反映 |
| 代表の稼働が限界に達する | 拡大停滞 | 教材化と非同期化、Tier 1 比重UP |
| 認知不足 | 受注ゼロ | ブログ・セミナーでインバウンド強化 |
| 受講後の自走化に失敗 | 評判悪化 | 卒業後3ヶ月フォロー、認定基準明確化 |
