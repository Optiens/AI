## エグゼクティブサマリー

1. ハーブ生産の段階的拡大（60m2 -> 120m2 -> 300m2）で年商360万円 -> 720万円 -> 1,800万円を試算
2. 高糖度イチゴは技術的ハードルと初期投資が極めて高く、2029年度時点では「試験栽培」に留めるのが現実的

## 1. 植物工場での高糖度イチゴ栽培

### 1.1 国内の主要事例

| 事業者 | 特徴 | 規模 | 糖度 |
|---|---|---|---|
| GRA（宮城県山元町） | ミガキイチゴブランド。IT活用の高品質イチゴ生産 | 大規模ハウス | 糖度14以上 |
| オイシイファーム（米国NJ） | 完全閉鎖型植物工場。AI受粉で糖度世界トップクラス | 大規模工場 | 糖度20以上 |
| HarvestX（東京大学発） | ロボット受粉（XV3）、成功率90%。浜松で商業化 | 中規模 | 非公開 |
| アクポニ + アグリ王 | アクアポニックス方式。通年栽培、無農薬 | 小〜中規模 | 糖度15（一般の1.5倍） |
| いちご研究室 | LED完全閉鎖型。研究開発段階 | 小規模 | 非公開 |

### 1.2 技術的要件

- **栽培方式**: NFT（薄膜水耕）またはDFT（深水流水耕）が主流
- **受粉**: ミツバチ導入が必須（閉鎖型ではロボット受粉 or 手動受粉）
- **環境制御**: 温度15-25度C、湿度60-80%、光量PPFD 200-400 umol/m2/s
- **高糖度化の鍵**: EC値を高めに管理（EC 1.5-2.5）して水分ストレスをかける
- **栽培期間**: 育苗から収穫終了まで12-24か月（回転効率が悪い）
- **IoTセンサーの活用**: EC/pH/水温/気温湿度の常時モニタリングが糖度安定化に直結

| 受粉システム（ミツバチ or ロボット） | 20-50万円/年 | 30-80万円/年 |
| IoTセンサー・制御系 | 既存設備転用可 | 追加30-50万円 |
| **初期投資合計** | **500-1,000万円** | **1,000-2,000万円** |
| 月間電気代 | 8-15万円 | 15-30万円 |
| 月間人件費（パート2名） | 30-40万円 | 50-70万円 |

**注意**: イチゴの植物工場は「黒字化に成功した事例がほとんどない」とイチゴテック社が指摘。大規模化しないと収益化が困難。

### 1.6 Optiensへの適用判断

- **2029年度時点**: 試験栽培（1棚程度）に留め、データ蓄積を優先

## 2. 高糖度ミニトマト（水耕栽培）

### 2.1 国内の主要事例

| 事業者・ブランド | 特徴 | 糖度 | 販売形態 |
|---|---|---|---|
| アメーラ（静岡/長野） | 水分制限栽培、通常の1/3サイズに凝縮 | 8度以上 | 1kg 2,500-4,000円（小売） |
| OSMIC（千葉） | 糖度11度以上のみ出荷。個別糖度測定 | 11度以上 | ギフト中心、1箱3,000-8,000円 |
| 信州トマト工房（長野） | 塩類ストレスによる高糖度化 | 10度以上 | 直販中心 |
| Happy Quality（静岡） | AIで糖度予測・品質管理 | 10度以上 | B2B中心 |

### 2.2 高糖度化の技術

| 手法 | 概要 | EC値目安 |
|---|---|---|
| **EC値制御（水分ストレス）** | 養液濃度を高めに維持し、水分吸収を制限。糖が凝縮される | EC 4.0-8.0（通常の2-4倍） |
| **塩ストレス** | 養液に塩化ナトリウムを添加。農研機構で効果実証済み | EC 8.0-12.0 |
| **低段密植栽培** | 草丈を低く抑え短期間で収穫。回転率を上げる | — |
| **日射制御** | 光量を調整し光合成を促進。糖の生成量を増やす | — |

**IoTとの親和性**: EC値・pH・水温・日射量のリアルタイムモニタリングが高糖度安定生産の核心技術。Optiensの既存IoTシステム（MCP3008 ADC + EC/pHセンサー + Zigbee制御）をそのまま転用可能。

### 2.3 販売単価・粗利率

| カテゴリ | 販売単価 | 粗利率（試算値） |
|---|---|---|
| 一般ミニトマト（市場出荷） | 500-800円/kg | 15-25% |
| 高糖度フルーツトマト（直販） | 2,000-4,000円/kg | 40-55% |
| 超高糖度ブランド（ギフト） | 5,000-10,000円/kg | 50-65% |
| EC通販（定期便） | 2,500-3,500円/kg | 45-55% |

### 2.4 年間売上規模の目安（試算値）

- 水耕栽培トマトの収量目安: 年間30-50kg/m2（多段式で最大80kg/m2）
- フルーツトマトは水分制限のため収量が通常の30-50%に低下

| 規模 | 年間収量 | 年間売上（直販3,000円/kg） |
|---|---|---|
| 20m2 | 200-500kg | 60-150万円 |
| 60m2 | 600-1,500kg | 180-450万円 |
| 120m2 | 1,200-3,000kg | 360-900万円 |

### 2.5 Optiensへの適用判断

- **2029年度から本格導入を推奨**
- **理由**: ハーブで確立したEC値制御・IoTモニタリング技術がそのまま活用可能。受粉不要（トマトは風・振動で自家受粉）。初期追加投資はハーブ用設備の転用で抑制可能
- **戦略**: 2028年度に試験栽培開始 -> 2029年度から1教室（20m2）で商業生産

## 5. 補助金申請書での「将来ビジョン」の書き方

### 5.1 審査員が見たいポイント

持続化補助金の審査では以下が重視される:

1. **売上・利益の数値目標**: 補助事業実施後1年・2年・3年の売上推移表を必ず含める
2. **雇用創出**: 「Phase 3でパート2名雇用、2029年度に3名へ拡大」など具体的に
3. **地域貢献**: 北杜市の廃校活用、地元飲食店との連携、地域ブランド向上
4. **差別化ポイント**: IoT×AI自動化による品質安定性、通年供給体制
5. **経営計画書と補助事業計画書の一貫性**: ストーリーが繋がっていることが必須

### 5.2 記述の推奨構成

```
[現状分析]
  北杜市は八ヶ岳エリアの観光飲食需要が高いが、
  地元産の新鮮ハーブの通年供給体制が不足している。

[補助事業の内容]
  EC構築・販促ツール整備により、
  販路拡大と認知度向上を図る。

[将来ビジョン（3-5年）]
  2027年度: ハーブ生産60m2で年商217万円、パート1名雇用
  2028年度: 120m2に拡大、エディブルフラワー追加、年商460万円、パート2名
  2029年度: 高糖度ミニトマトを追加し品目多角化、年商520万円、パート2-3名
  2030年度以降: 300m2規模に拡大、年商1,000万円超を目指す

[地域への波及効果]
  - 廃校の有効活用（北杜市の遊休公共施設問題の解消に貢献）
  - 地元レストランへの新鮮食材供給（地産地消の推進）
  - IoT農業のモデルケースとして他の新規就農者への技術波及
  - 雇用創出（2029年度までにパート3名）
```

### 5.3 記述上の注意点

- **数値は控えめに**: 楽観的すぎる数値は審査員の不信を招く。上記試算は既に60%シナリオだが、補助金申請書では更に保守的な数値を使うことを推奨
- **イチゴは「検討中」に留める**: 初期投資が大きく実現性が低いと判断される恐れがあるため、申請書には「将来的な検討課題」程度の言及に留める
- **トマトは「既存技術の応用」として記述**: ハーブ栽培で確立したEC値制御技術を高糖度トマトに応用する、という論理展開が説得力を持つ
- **「仕組み化」を強調**: 「農業システムの設計者」として、IoT×AIで再現性のある生産体制を構築する点を差別化要因として強調

## 7. 結論と推奨事項

### 補助金申請書への記載推奨事項

1. **2029年度ビジョンの中心は「ハーブ品目多角化 + 高糖度ミニトマト追加」**
   - イチゴは技術・投資のハードルが高すぎるため、申請書の主軸にしない
   - ミニトマトはEC値制御技術の延長線上にあり、説得力が高い

2. **売上目標は保守的に設定**
   - 2029年度: 年商500万円（本レポート試算520万円の保守寄り）
   - 2030年度: 年商800-1,000万円

3. **地域貢献・雇用創出を数値で示す**
   - 2029年度までにパート雇用3名（地元雇用）
   - 廃校教室3-4室の有効活用
   - 地元飲食店10件以上との取引

4. **IoT技術の横展開可能性に言及**
   - 栽培データの蓄積 -> 他品目への応用 -> 生産品目の段階的拡大
   - 「農業システムの設計者」としての独自性
