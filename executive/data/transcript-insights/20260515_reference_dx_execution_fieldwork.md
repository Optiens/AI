## 中核メッセージ

DXやAI導入は、流行のツールを入れることではなく、利益・生産性・競争力に返る業務の仕組みを作ること。目的設定、現状業務の詳細な分解、課題抽出、投資対効果に応じた解決策、現場定着までを一連で扱わないと、使われないシステムや形だけのワークショップで終わりやすい。

## Optiensへの適用ポイント

- **有償診断の評価基準**: AI化候補は「できるか」ではなく「どの業務課題を、いくらの投資で、どれだけ改善できるか」で評価する。
- **レビュー面談の価値訴求**: 30,000円前後のAI活用レビュー面談では、いきなり実装を売るのではなく、To-Be / As-Is / ROI / 現場定着リスクを切り分ける。
- **提案書の表現**: 「AIで自動化できます」では弱い。提案には、現行作業、削減対象、残す人間作業、初期実装範囲、運用定着の方法を入れる。
- **実装範囲の制御**: 10万円分の課題に100万円の仕組みを提案しない。50%改善、70%改善、手作業併用など複数案で出す。
- **AI診断レポートの改善余地**: 「業務フローが粗い企業ほど、まずマニュアル化・入力フォーマット整理が効く」ことを診断コメントに反映できる。

## 採用しない要素

- 元動画の会社名、代表者名、チャンネルの合言葉、ダウンロードCTAは採用しない。

- 「DXは儲けるためだけ」とは書かず、公式定義に合わせて「競争優位・企業価値・収益性に接続する経営課題」と表現する。
