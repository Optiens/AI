# 水耕栽培IoTシステム 機器比較調査

## 調査目的

Phase 1（自宅テスト栽培）およびPhase 2（廃校教室20m2）で使用する機器の選定根拠を整理する。現在保有機器の継続利用可否と、アップグレード候補の比較を行う。

---

## 1. メインコンピュータ: Raspberry Pi 4B vs Pi 5 vs ミニPC

### 比較表

| 項目 | RPi 4B 4GB（保有済） | RPi 5 8GB | ミニPC（Intel N100系） |
|---|---|---|---|
| **価格** | 0円（保有済×2台） | 約14,000-16,000円 | 約20,000-30,000円（16GB/512GB SSD） |
| **CPU性能** | Cortex-A72 1.5GHz 4コア | Cortex-A76 2.4GHz 4コア（Pi4の2-3倍） | Alder Lake-N 3.4GHz 4コア（Pi5の1.5-2倍） |
| **RAM** | 4GB LPDDR4 | 8GB LPDDR4X | 8-16GB DDR4/DDR5 |
| **ストレージ** | microSD（故障リスク有） | microSD / NVMe HAT | eMMC / M.2 SSD内蔵（高信頼） |
| **GPIO** | 40ピンヘッダ有 | 40ピンヘッダ有 | なし（USB変換が必要） |
| **SPI/I2C** | ネイティブ対応 | ネイティブ対応 | USB-SPI/I2Cアダプタ経由 |
| **消費電力（アイドル）** | 約3-4W | 約4-5W | 約6-10W |
| **消費電力（負荷時）** | 約6-7W | 約8-12W | 約15-25W |
| **発熱** | ヒートシンク推奨 | ファン/アクティブクーラー必須 | 内蔵ファン有（ファンレスモデルも有） |
| **OS** | Raspberry Pi OS（Debian系） | Raspberry Pi OS（Debian系） | Ubuntu/Debian/Windows |
| **OSサポート期間** | 2028年頃まで（Pi OS更新） | 2030年以降 | Ubuntuなら5年LTS |
| **安定性** | SDカードが弱点 | NVMe対応で改善可 | SSD内蔵で高い |
| **代表機種** | - | Raspberry Pi 5 8GB | GMKtec NucBox G3, Beelink Mini S12 Pro, MINISFORUM UN100C |

### SDカード故障問題の実態

| 条件 | 想定寿命 |
|---|---|
| 軽負荷（読み取り中心・ログ少量） | 3-5年 |
| 中負荷（MQTT・DB書き込み有） | 1-2年 |
| 高負荷（DB頻繁書き込み・ログ大量） | 数ヶ月-1年 |
| USB SSD起動に変更した場合 | 5-10年以上 |

**故障の主因**: SDカードはカメラ・スマホ用に設計されており、OSの継続書き込みに対する耐性が低い。特に停電時のデータ破損が致命的。SanDisk Endurance/Samsung PRO Enduranceなどの監視カメラ用カードで改善可能だが根本解決にはならない。

**推奨対策**:
- RPi 4BをUSB SSD起動に変更（コスト: SSD 128GB 約2,000-3,000円 + USB-SATA変換 約1,000円）
- ログ書き込み先をRAMdiskにし、定期的にflush
- `/var/log`をtmpfsにマウント

### ミニPCでGPIOがない場合の対策

| 方式 | 製品例 | 価格 | 特徴 |
|---|---|---|---|
| USB-SPI/I2C/GPIOアダプタ | Adafruit FT232H | 約3,980円（Amazon.co.jp） | SPI/I2C/GPIO対応、Python(pyftdi)対応 |
| USBシリアル + マイコン | ESP32-DevKit | 約1,500-2,000円 | WiFi/BLE内蔵、ADC/GPIO豊富、MQTTでブリッジ |
| USB-I2Cアダプタ | MCP2221A搭載ボード | 約1,000-2,000円 | I2C + UART、シンプル |
| 独立マイコンセンサーノード | ESP32 + センサー群 | 約3,000-5,000円一式 | WiFi経由MQTT、ミニPCと完全分離 |

**推奨アーキテクチャ（ミニPC採用時）**:
```
[ミニPC] --- MQTT/WiFi --- [ESP32センサーノード]
  |                              |
  Zigbee2MQTT                    センサー（EC/pH/温湿度等）
  Supabase連携                   アクチュエータ制御
  AI推論
```

### 判定

| 用途 | 推奨 | 理由 |
|---|---|---|
| Phase 1（自宅テスト） | **RPi 4B（保有済）+ USB SSD化** | 追加コスト最小（約3,000-4,000円）、GPIO直結 |
| Phase 2（20m2生産） | **RPi 5 or ミニPC** を検討 | Pi4は性能限界、AI推論にはN100有利 |
| Zigbee2MQTTゲートウェイ | **RPi 4B（Pi1）** | Zigbee USBドングルとの相性が確立済み |
| センサーデータ収集 | **RPi 4B（Pi2）+ USB SSD** | GPIO直結、現行システム維持 |

| フェーズ | 推奨 | 理由 |

## 4. ペリスタルティックポンプ（養液精密投入）

### 比較表

| 項目 | 中国製DC12Vポンプ | Kamoer KAS（ステッピング） | Kamoer KCM-ODM | DFRobot Gravity デジタルポンプ |
|---|---|---|---|---|
| **モーター** | DCブラシモーター | ステッピングモーター | ステッピングモーター | サーボ制御 |
| **流量** | 約0-100ml/min（一定） | 5-84ml/min（可変） | 10-452ml/min（可変） | 0.5-40ml/min |
| **精度** | 低（ON/OFFのみ） | 高（ステップ制御で精密） | 高 | 中-高（PPMサーボ制御） |
| **制御方式** | リレーON/OFF | ステッピングドライバ（GPIO PWM） | ドライバボード付属 | PWMサーボ信号 |
| **RPi接続** | GPIO → リレー → ポンプ | GPIO → ステッピングドライバ → ポンプ | USB or ドライバボード | GPIO PWM直結 |
| **電源** | 12V DC | 12V DC | 12V DC | 5V DC（RPiから給電可） |
| **価格** | 約1,000-2,000円（Amazon/AliExpress） | 約5,000-8,000円 | 約10,000-15,000円 | 約3,000-5,000円 |
| **チューブ寿命** | 短（硬化しやすい） | シリコンチューブ交換可能 | BPTチューブ長寿命 | シリコンチューブ |
| **用途適合** | 粗い投入（循環ポンプ代用） | A液/B液精密投入に最適 | 大流量が必要な場合 | 小規模精密投入 |

### 必要本数（養液自動管理）

- A液ポンプ x 1
- B液ポンプ x 1
- pH Down（リン酸）ポンプ x 1
- pH Up（水酸化カリウム）ポンプ x 1（任意）
- 合計: 3-4本

### RPiからの制御方法

```
方式1: GPIO → L293Dモータードライバ → DCモーターポンプ
  - 速度制御: PWM
  - 方向制御: H-Bridge
  - コスト: ドライバ約500円

方式2: GPIO → A4988ステッピングドライバ → ステッパーポンプ
  - ステップ制御で正確な投入量
  - DIR/STEPピンでGPIO2本使用
  - コスト: ドライバ約300-500円

方式3: PWMサーボ信号 → DFRobotポンプ
  - RPi PWM出力で直接制御
  - 最もシンプルだが精度は方式2に劣る
```

### 判定

| フェーズ | 推奨 | コスト（3本セット） |
|---|---|---|
| Phase 1 | **中国製DCポンプ + L293D** でプロトタイプ | 約5,000-7,000円 |
| Phase 2 | **Kamoer KAS（ステッピング）** | 約15,000-24,000円 |
| Phase 3 | **Kamoer KCM-ODM or 産業用** | 約30,000-45,000円 |

## 調査における注意事項

1. **価格は2026年3月時点の概算**。為替変動・在庫状況で変わる
2. **Atlas Scientificは日本の正規代理店が限られる**。DigiKey Japan経由が確実
3. **HLGは日本Amazon取り扱いなし**。個人輸入の場合は送料・関税に注意
4. **中国製LED（AliExpress）はスペック詐称が多い**。実測PPFDが公称の50-70%という報告多数
5. **ペリスタルティックポンプのチューブは消耗品**。シリコンチューブは3-6ヶ月、BPTチューブは1年程度で交換推奨
