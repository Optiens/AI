# 月次レビューフォーマット v1.0

**作成日**: 2026-05-13
**対象**: 合同会社Optiens 月次経営レビュー（毎月初実施）

毎月、先月分の数字をこのフォーマットで振り返る。所要時間 15-20 分。

---

## 使い方

1. 月初（1〜5日）に `/monthly-accounting-check` Skill を実行
2. 出力結果を以下のフォーマットに転記
3. `executive/finance/monthly-review/YYYY-MM.md` として保存
4. 異常があれば即対応、なければ次月へ

---
