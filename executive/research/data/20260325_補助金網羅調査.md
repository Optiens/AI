## 新たに発見された有望な補助金

### 1. みどりの食料システム法認定 — 適合度B（Phase 2推奨）
- 植物工場（水耕栽培）は認定を取得しやすい
- 2026年度から農水省の全補助金で環境負荷低減（クロスコンプライアンス）が必須化
- 早期認定で今後のすべての農水省系補助金で優先採択される
- 税制優遇: 機械32%・建物16%の特別償却
- **申請**: 都道府県に5カ年の環境負荷低減事業計画を提出（随時）

### 2. デジタル化・AI導入補助金2026 — 適合度B
- freee会計、食べチョク等の既製ITツール導入であれば申請可能
- 通常枠: 5万〜450万円（補助率1/2）
- **1次締切: 2026/5/12** — 持続化補助金直後に申請可能
- 持続化=販促費、デジタル化=ITツール導入費と分ければ併用可能
- **制約**: 登録ITツールのみ対象。自作IoTシステムは対象外

### 3. 北杜市チャレンジ農業助成金 — 適合度B
- 資材費等の1/2以内（最大100万円）
- 高単価ハーブ生産は「高収益農業」に該当する可能性
- 国の補助金と市の助成金は原則併用可能
- **要確認**: 認定農業者/認定新規就農者の認定要否（0551-42-1350に電話）

### 4. 6次産業化支援 — 適合度B（Phase 2-3）
- 生産→加工（ドライハーブ等）→販売（EC・直販）の6次産業化モデルに該当
- 認定自体は無料。融資特例・6次産業化プランナーの無料助言
- Phase 2-3で加工品展開時に有効

### 5. 山梨県IoT補助金 — 適合度B
- IoTトライアル事業: 補助率1/2、上限25万円
- IoTモデル事業: 補助率1/2、上限100万円
- RPi・センサー・カメラ等の導入費が補助対象に合致
- **要確認**: 2026年度公募の有無（055-223-1534に電話）

## 不適合と判断したもの

| 補助金 | 不適合理由 |
|---|---|
| ものづくり補助金 | 従業員0名では申請不可。Phase 2で雇用後に再検討 |
| 経営開始資金 | 世帯所得600万円要件でほぼ不適合（SF年収720万円） |
| 新事業進出補助金 | 創業1年未満は対象外。2027年4月以降に検討 |
| 北杜市創業促進支援 | 農業は対象外業種 |
| 強い農業づくり交付金 | 産地単位の大規模事業向け。現フェーズでは不適合 |
| NEDO SBIR | 技術開発特化。農業生産事業には不向き |

## 併用ルール

| 組み合わせ | 可否 | 備考 |
|---|---|---|
| 持続化 + デジタル化・AI導入 | **可** | 持続化=販促、デジタル化=ITツール |
| 持続化 + ものづくり | **可** | 持続化=販促、ものづくり=設備 |
| 持続化 + 北杜市チャレンジ農業 | **可** | 国+市、経費を分ける |
| 持続化 + 山梨県IoT補助金 | **可** | 国+県、経費を分ける |
| 持続化（一般型）+ 持続化（創業型） | **不可** | 同一制度の重複 |
| 持続化 + みどり認定税制 | **可** | 補助金+税制優遇 |
