# Optiens 法務・契約書類テンプレート（DRAFT v0.1）

## ⚠️ 重要：本番利用前に必ず弁護士確認を受けること

本ディレクトリの全テンプレートは、CEO 補佐用のドラフトです。
**法的有効性・税務的妥当性は弁護士・税理士による確認が完了するまで未確定**です。

DoR（Definition of Ready）項目：
- [ ] 業務委託契約書（要件定義）テンプレ完成（弁護士確認済）
- [ ] 業務委託契約書（導入支援）テンプレ完成（弁護士確認済）

- [ ] 充当時の消費税処理を税理士確認済
- [ ] 不成約時の成果物（要件定義書）の著作権帰属確定

## 運用ルール

### バージョン管理
- DRAFT v0.x: 弁護士未確認の社内ドラフト
- v1.0: 弁護士確認後の正式運用版（DoR充足）
- 改訂時は `_vX.Y` でバージョンアップ

### 顧客への提示
- 弁護士確認前は「Phase A1 暫定運用」として CEO 判断で運用可（業務フロー定義 DoR 暫定運用ルール準拠）
- 暫定運用中の案件は、契約書テンプレ完成後に **遡って正式契約書で再締結する** ことを顧客と合意しておく

### インボイス対応
- 合同会社Optiens の適格請求書発行事業者番号: **登録申請中**（取得後に各テンプレを更新）
- 取得まで「適格請求書発行事業者 申請中」と記載

### 印紙税
- 業務委託契約書（請負契約）は印紙税の対象になる場合あり
- 電子契約（PDF メール送付・電子署名）の場合は印紙不要
- 紙ベースの場合の印紙額は税理士確認
