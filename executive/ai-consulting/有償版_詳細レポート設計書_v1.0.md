# 有償版 詳細レポート設計書 v1.8

## 1. 現行定義

詳細版AI活用診断は、税込5,500円の有償詳細レポートである。
60分MTGは含めない。レポート内容の解説や導入支援前の相談が必要な場合は、AI活用レビュー面談または導入支援相談として別導線で扱う。

### 価格・納品

| 項目 | 定義 |
|---|---|
| 価格 | ¥5,500（税込） |
| 納品物 | Google Slides URL |
| 納期 | 入金確認後5営業日以内 |
| MTG | なし |
| 充当 | 導入支援契約に進む場合、詳細版費用を初期費用へ全額充当 |
| 返金 | 納品前キャンセル、または納品後7日以内の内容不備・不満申出に対応 |

### 無料版との差

| 項目 | 簡易版 | 詳細版 |
|---|---|---|
| 価格 | 0円 | ¥5,500税込 |
| 目的 | 方向性と費用帯の一次判断 | 導入可否、優先順位、費用前提の判断 |
| 主情報源 | 基本フォーム、画像/URL入力補助 | 詳細フォーム、画像/URL入力補助、追加自由記述 |
| 提案数 | TOP3中心 | 7件 |
| 成果物 | 簡易Google Slides | 19スライド詳細Google Slides |
| 個別相談 | 別導線 | 別導線 |

## 2. 添付ファイル対応の定義

「詳細版の添付ファイル対応」とは、顧客がフォームに入力する代わりに、手元資料や自社サイトから情報を読み取ってフォームを補完する機能を指す。

現行実装では、フォーム内の直接アップロードは画像のみ対応する。

| 入力方法 | 現行対応 | レポートへの扱い |
|---|---|---|
| 画像アップロード | JPG / PNG / GIF / WebP、最大5MB | AI読取後、フォーム項目の補完情報として利用 |
| URL読取 | 公開ページURL | AI読取後、フォーム項目の補完情報として利用 |
| PDF / Excel / Word | 現行フォームでは直接添付しない | 必要な内容は自由記述に要約するか、別相談で扱う |

このため、公開表現では「PDF・Excel・Word対応」と書かない。
詳細版の品質は、添付ファイルだけで担保するのではなく、詳細フォーム回答と画像/URL補助の組み合わせで担保する。

## 4. レポート生成フロー

現行実装は `supabase/functions/process-paid-diagnosis/index.ts` を正とする。

1. `src/pages/free-diagnosis.astro` で詳細版を選択し、フォーム送信
2. `src/pages/api/free-diagnosis.ts` が `plan='paid'` のリードを作成し、振込案内メールを送る
3. 入金確認後、リードの `status` を `paid` にする
4. `process-paid-diagnosis` が詳細レポートJSONを生成する
5. 機械品質ゲートで未置換プレースホルダー、禁止語、施策欠落、数値根拠不足を検出する
6. Google Slidesテンプレートをコピーし、`{{paid_xxx}}` プレースホルダーを置換する
7. 顧客へGoogle Slides URLをメール送付し、`status='report_sent'` にする
8. 失敗時は `manual_review` に落とし、管理者へ通知する

## 5. 品質基準

### 必須

- 7件の提案を出す
- 各提案に優先度、削減時間、月間効果額、根拠、前提条件を入れる
- 優先度Aの上位3件は導入前後フローと構成案まで示す
- 概算導入費は安売りしない
- リスクは「発生可能性: 中」を並べず、対策込みの管理方針として書く
- 補助金は候補と確認ポイントにとどめ、申請支援を約束しない
- レポートがAI自動生成・品質ゲート通過後に自動送付される前提を崩さない

### 費用レンジ

| 区分 | 目安 |
|---|---|
| 1業務MVP | ¥270,000〜360,000 |
| 2業務実装 | ¥420,000〜560,000 |
| 3業務以上 / 外部連携多め | ¥600,000〜800,000+ |

正式見積は、導入支援のご相談時に対象業務、連携先、データ状態、運用体制を確認して確定する。

## 6. 詳細版で書かないこと

- 詳細レポートに60分MTGが含まれるという表現
- PDF / Excel / Wordを現行フォームで直接添付できるという表現
- 「人が確認して作成した」と誤解される表現
- 成果保証、回収期間の断定
- IT導入補助金の申請支援を行うような表現
- AGENTS.mdの禁止事項に該当する旧方針
