# 水耕栽培・エアロポニクス対応 高単価植物 網羅調査レポート

**調査日**: 2026年3月24日
**目的**: Phase 3以降の品目拡大候補を網羅的に評価し、優先順位を付ける
**前提条件**: 山梨県北杜市・標高1200m・冷涼気候・廃校教室（室内人工光型）・IoT×AI制御

---

## エグゼクティブサマリー

1. **最有望**: ワサビ（水耕）、サフラン（室内垂直農法）、高麗人参（水耕）は超高単価だが栽培難易度が高く長期投資が必要
2. **中期導入推奨**: アイスプラント、クレソン、ベビーリーフミックス、スプラウト類はハーブ設備を転用でき追加投資が少ない
3. **エアロポニクス特化**: 種芋（ジャガイモ）生産は世界的に実績があり、ニッチだが高収益。サツマイモ苗も有望
4. **非推奨**: バニラ（受粉問題・3年待ち）、メロン（設備投資大）、切り花（光量・温度管理の負荷大）は現段階では見送り
5. 既存計画（バジル・マイクログリーン・ディル・ミント・エディブルフラワー・高糖度ミニトマト）は最も合理的な品目構成であり、変更不要

## 凡例

- **水耕/エアロ相性**: ◎=最適 / ○=良好 / △=条件付き / ×=不向き
- **栽培難易度**: ★=初心者 / ★★=中級 / ★★★=上級
- **IoT親和性**: 高/中/低（センサー制御による品質向上効果の度合い）
- **北杜市適性**: ◎=好適 / ○=問題なし / △=要対策 / ×=不適
- 単価・収量は全て試算値。出典が明記されていない数値は複数ソースの平均値

## 1. 薬用植物・メディカルハーブ

### 1.1 ワサビ（本わさび）

| 項目 | 内容 |
|---|---|
| 水耕相性 | ○（技術確立途上。金沢工大が露地の3-4倍の生育速度を実証） |
| エアロ相性 | △（根茎肥大にはNFT/DFTが適する） |
| 販売単価 | 卸値: 15,000-27,000円/kg（本わさび根茎）、葉わさび: 3,000-5,000円/kg |
| m2年間売上（試算値） | 30,000-80,000円（根茎1.5-2年サイクル、多段不可のため面積効率低い） |
| 栽培難易度 | ★★★（水温8-18度C維持が必須、高温に極端に弱い） |
| 初期追加投資 | 冷却装置（チラー）が必須: 20-50万円 |
| IoT親和性 | **高**（水温・EC・pH・溶存酸素の精密制御が品質に直結） |
| 北杜市適性 | ◎（冷涼気候は本わさびに最適。天然水も豊富） |
| 国内事例 | 金沢工業大学コンテナ型ワサビ工場（2023年稼働）、六洋電気（福島・飯舘村）、NEXTAGE わさび栽培モジュール |
| 備考 | 日本の生産量が過去20年で70%減少。供給不足で価格上昇中。北杜市の冷涼環境はワサビに理想的だが、収穫まで1.5-2年（水耕で短縮可能性あり）。葉わさびなら3-4ヶ月で収穫可能 |

**判定: Phase 3後半（2028年度）から試験栽培推奨。葉わさびを先行させ、根茎は長期プロジェクトとして位置付ける。**

### 1.2 高麗人参（オタネニンジン）

| 項目 | 内容 |
|---|---|
| 水耕相性 | ○（韓国・日本で水耕栽培の特許技術あり。120日短期栽培法が確立） |
| エアロ相性 | ○（根の成長にエアロポニクスが有効との研究あり） |
| 販売単価 | 乾燥根: 110,000-130,000円/kg（$500-600/lb）。生根: 30,000-50,000円/kg |
| m2年間売上（試算値） | 50,000-150,000円（短期水耕栽培120日、サプリメント原料向け） |
| 栽培難易度 | ★★★（直射日光NG、温度15-25度C、遮光・湿度管理が重要） |
| 初期追加投資 | 遮光設備・専用水耕ラック: 10-30万円 |
| IoT親和性 | **高**（温度・湿度・光量の精密制御がジンセノサイド含有量に直結） |
| 北杜市適性 | ◎（冷涼気候に適合。低温期間が根の品質向上に寄与） |
| 国内事例 | 福岡県の水耕栽培高麗人参サプリメント（QVC販売）、アクアポニックス高麗人参（日本唯一を謳う事業者）、DTS株式会社（ミネラル調整法による水耕栽培） |
| 備考 | 露地栽培は4-6年かかるが、水耕なら120日で収穫可能（葉・茎含む）。サプリメント原料としての出口が安定。ただし薬事法に注意が必要 |

**判定: 非常に高単価だが販路確保（サプリメントメーカーとの契約）が前提。Phase 3以降の研究開発テーマとして検討。**

### 1.3 サフラン

| 項目 | 内容 |
|---|---|
| 水耕相性 | ○（球根の水栽培は容易。ただし球根の再肥大が課題） |
| エアロ相性 | △（球根作物のためエアロポニクスは不向き） |
| 販売単価 | 乾燥めしべ: 500,000-1,500,000円/kg（世界最高額スパイス）。国産は更にプレミアム |
| m2年間売上（試算値） | 10,000-50,000円（収量が極めて少ない: 約8.6g/m2/年。150球/m2で年間収穫） |
| 栽培難易度 | ★★（栽培自体は容易だが、収穫が超労働集約的。1kgのサフランに約15万本の花が必要） |
| 初期追加投資 | 球根代: 球根1個100-300円×密度150個/m2 = 15,000-45,000円/m2 |
| IoT親和性 | 中（温度・光周期の制御で開花タイミング制御。年最大9サイクル可能との報告あり） |
| 北杜市適性 | ◎（冷涼気候は球根の休眠打破に好適） |
| 国内事例 | 大分県竹田市（国内最大産地）。室内垂直農法の事例は国内にはまだ少ない |
| 海外事例 | Saffron Tech（イスラエル）、BlueRedGold（スウェーデン・273万ユーロ調達）、Veles Farming（スロバキア・80%自動化） |
| 備考 | kg単価は世界最高だがm2あたり収量が極めて少なく、面積効率はハーブに劣る。球根の調達・再利用サイクルの確立が鍵。イタリアの事例ではコンテナ農場で年9サイクル達成 |

**判定: ロマンはあるが、m2効率がハーブの1/3以下。小規模（1-2棚）で趣味的に始めるなら可。本格商業化は困難。**

## 12. 結論と推奨アクション

### 即時アクション（Phase 2開始時）

1. **クレソン・ルッコラを既存ハーブラインに追加**（追加投資ゼロ、種代のみ）
2. **ブロッコリースプラウト・そばスプラウトをマイクログリーンラインに追加**（追加投資ゼロ）
3. 「北杜市産そばスプラウト」のブランディング検討

### 中期アクション（Phase 2後半-Phase 3）

4. **アイスプラントの試験栽培開始**（EC値制御技術の応用）
5. わさび菜・シソの追加（和食レストラン向け販路開拓とセット）
6. ヒマワリスプラウトの試験栽培

### 長期アクション（Phase 3以降）

7. **葉わさびの試験栽培**（チラー導入が前提。北杜市の冷涼環境を最大活用）
8. 高麗人参の研究栽培（サプリメント原料としての出口調査を並行）
9. エアロポニクスシステムの実験区設置（種芋生産の可能性検証）
10. 本わさび根茎の長期栽培プロジェクト開始

### 見送り品目（再検討条件付き）

- **バニラ**: 北杜市での栽培は非現実的。将来的に温室施設を得た場合のみ再検討
- **メロン**: 太陽光利用型施設への移行時に再検討
- **切り花**: 同上
- **サフラン**: 趣味的栽培は可。商業化は球根コストと収量の問題が解決された場合のみ
- **甘草**: 水耕技術が確立された場合のみ
