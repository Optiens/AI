# 就労支援 x 水耕栽培 市場・競合・AI活用調査

## 調査目的

就労継続支援事業所（A型/B型）における水耕栽培・植物工場の導入実態、IoTシステム販売の競合状況、Tesla Optimusの最新動向、AI活用による農業人件費削減の可能性を調査する。

### 1.2 導入システムの傾向

- **LED照明**: 閉鎖型植物工場はほぼ全施設がLED採用。アグリ王「AGRI Oh!」やキーストーンテクノロジー製が多い
- **水耕方式**: NFT（薄膜水耕）、DFT（湛液水耕）が主流。ミスト式は少数
- **環境制御**: 温度20-24度C、湿度60-80%に制御。CO2濃度管理は一部施設のみ
- **IoT導入**: 大半の施設は**手動管理**。IoT・センサーによる自動制御を導入している施設はごく少数
- **作業の細分化**: 播種→育苗→定植→収穫→梱包→清掃に工程を分割し、障がい特性に合わせた配置

### 1.3 成功要因と失敗リスク

**成功要因:**
- 天候に左右されない安定した室内作業環境
- 障がい特性に合わせた作業配分が容易
- 完全無農薬を訴求でき、販路確保しやすい
- 訓練等給付費による安定収入が運営を下支え

**失敗リスク・課題:**
- 植物工場全体の**約75%が赤字**（農水省調査）。福祉事業所でも例外ではない
- 電気代（LED+空調）が最大のコスト要因。燃料費高騰で一層深刻化
- 葉物野菜は市場価格が低く、露地栽培との価格競争に弱い
- 人材確保が困難（専門知識を持つ職業指導員・サービス管理責任者の不足）
- 初期投資が重い（小型でも数百万円、中規模で1,000万円超）

### 1.4 訓練等給付費の仕組み

訓練等給付費は、国（50%）・都道府県（25%）・市町村（25%）が負担する障害福祉サービスの公費報酬。

#### B型の報酬単価（令和6年度改定後）

| 区分 | 基本報酬（単位/日） | 金額換算（10円/単位、地域加算前） |
|---|---|---|
| 平均工賃月額4.5万円以上（定員20人以下） | 702単位 | 約7,020円/日 |
| 平均工賃月額1万円以上（定員20人以下） | 611単位 | 約6,110円/日 |
| 平均工賃月額5千円以上（定員20人以下） | 590単位 | 約5,900円/日 |
| 平均工賃月額5千円未満（定員20人以下） | 566単位 | 約5,660円/日 |

**1人あたりの月額収入（B型、典型例）:**
- 日額約6,100-6,500円 x 月22日稼働 = **約13.4万-14.3万円/人/月**
- 地域加算（1級地で11.14円/単位）を含めると**約15万円/人/月**
- 各種加算（処遇改善加算、福祉専門職員配置等加算など）で+10-20%

#### A型の報酬単価

- A型は1日あたり約5,900円が目安（令和6年度改定でスコア方式見直し）
- 生産活動収支・平均労働時間による評価
- 月額では**約13万円/人/月**程度

#### 収益構造のポイント

- 定員20人のB型事業所の場合、稼働率80%で: 16人 x 14万円 = **月額約224万円**の訓練等給付費
- これに生産品の販売収入が上乗せ
- 一方、人件費（サービス管理責任者1名+職業指導員+生活支援員）・家賃・光熱費・設備償却が固定費

### 2.3 市場の飽和度評価

**結論: ニッチだが「就労支援 x IoT」は未開拓領域が大きい**

- 「LED植物工場 x 福祉」の導入支援はアグリ王・キーストーンテクノロジーがほぼ独占
- しかし、**IoTセンサー・自動制御・データ分析を含むパッケージ**を福祉施設に提供している企業は**ほぼ皆無**
- 大半の福祉施設は手動管理。「IoTで作業負荷を軽減し、障がい者でも管理できるシステム」への需要は潜在的に大きい
- 農園型雇用代行（エスプール、スタートライン）は「雇用枠の代行」であり、IoTシステム販売とは異なるビジネスモデル

**注意: Optiens社の現在の事業方針は「業務用ハーブ生産」であり、就労支援事業所へのシステム販売は事業範囲外。本調査は市場理解のための参考情報。**

## 3. Tesla Optimus（オプティマス）最新状況

### 3.1 開発進捗（2026年3月時点）

| 項目 | 詳細 |
|---|---|
| 最新世代 | **Optimus Gen 3**（2026年Q1に動画公開済み） |
| 身長/体重 | 5'8"（173cm）/ 約57kg |
| 手の自由度 | 片手22DoF+手首・前腕3DoF = **片手25アクチュエータ**（Gen 2の4.5倍）|
| AI | Grok（xAI製LLM）統合。観察学習・リアルタイム行動適応 |
| 現在の状態 | テスラ社内で歩行・洗濯などのタスクを実行中。**「useful work」は未達成**（Q4 2025決算） |
| 生産拠点 | フリーモント工場（Model S/X生産終了→Optimus生産ラインに転換） |
| 量産目標 | 年間100万台（長期目標） |

### 3.2 価格予測

| 時期 | 価格帯 | 備考 |
|---|---|---|
| 2026年後半（限定販売） | 約$30,000（約450万円） | プレミアム価格。限られた外部顧客向け |
| 2027年（一般販売開始予測） | $20,000-$30,000（300-450万円） | マスク氏目標 |
| 長期目標 | $20,000未満（300万円未満） | 量産効果が前提 |
| 現在の製造コスト | $50,000-$100,000推定 | 量産前の段階 |

### 3.3 日本での利用可能時期

- テスラは**日本市場への具体的なOptimus投入時期を未発表**
- 米国での限定販売が2026年後半、一般販売が2027年予定
- 日本への展開は**早くても2028年以降**と推測される（各国規制対応・安全認証が必要）
- 産業用ロボットとしての日本の安全規格（JIS B 8433等）への適合が前提条件

### 3.4 農業・水耕栽培での活用可能性

**現時点での評価: 非現実的**

- Gen 3は「汎用ヒューマノイド」だが、農業特化の設計ではない
- 手の自由度は向上したが、繊細なハーブの収穫（バジルの葉を傷つけずに摘む等）には未検証
- 高温多湿・水場での動作保証なし（植物工場は湿度60-80%）
- 300-450万円/台は、同等の作業を人間のパートで年間100-150万円で代替可能
- **Optiens社のPhase 2（廃校教室20m2）規模では投資対効果が全く合わない**
- 産業用途（ピッキング、搬送）で実績を積んだ後、農業展開は2030年代以降か

## 4. AI活用による農業人件費削減

### 4.1 AIで自動化できる農業タスク

| タスク | 自動化手段 | 実現度（2026年時点） |
|---|---|---|
| **環境モニタリング** | IoTセンサー+クラウドダッシュボード | 実用化済み |
| **養液管理** | EC/pHセンサー+自動補正（ペリスタルティックポンプ） | 実用化済み |
| **照明制御** | タイマー+光量センサー+AI最適化 | 実用化済み |
| **収穫量予測** | カメラ画像+機械学習 | 一部実用化 |
| **病害虫検出** | 画像認識AI | 一部実用化 |
| **作業指示・判断支援** | LLM（Claude等）によるデータ解釈・アドバイス | 実験段階 |
| **生育状態のNL報告** | MCP経由でセンサーデータをClaudeが解釈→自然言語で報告 | Optiens社で構築中 |
| **収穫作業** | 自動収穫ロボット | 大規模施設のみ |
| **播種・定植** | 自動播種機 | 大規模施設のみ |

### 4.2 日本の先進事例

| 企業 | 取り組み | 効果 |
|---|---|---|
| **スプレッド**（京都） | テクノファームけいはんな。AIで栽培データから最適条件算出・収量予測 | 世界最大規模の自動化植物工場の安定稼働 |
| **デンソー** | 低コスト自動収穫ロボット | 収穫量20%以上改善、パート人件費50%削減を目標 |
| **サラダボウル** | レール搬送システム（作物が作業者のもとに来る） | 労務コスト大幅削減。同面積の露地比3-5倍の収穫量 |

### 4.3 現実的に削減できる割合

**小規模水耕栽培（Phase 1-2）の場合:**

| 領域 | 削減可能な割合 | 手段 |
|---|---|---|
| 環境モニタリング | 90-100% | IoTセンサー自動化（Optiens社で構築中） |
| 養液管理 | 70-80% | EC/pH自動補正。異常時のみ人間介入 |
| 照明制御 | 100% | タイマー+Zigbee。完全自動化可能 |
| 収穫作業 | 0% | 手作業が必須（小規模では自動化不採算） |
| 播種・定植 | 0-10% | 手作業中心。将来的にはジグ化で効率改善 |
| 梱包・出荷 | 0% | 手作業 |
| 判断・意思決定 | 50-70% | Claude Code+MCPでデータ解釈→提案。最終判断は人間 |
| **全体** | **30-40%** | 管理業務の自動化が中心。物理作業は人間依存 |

### 4.4 LLMの農業分野への適用限界

Anthropicの労働市場影響レポート（2026年2月）によると:
- 農業はAIカバレッジが**15.7%**と全産業中で最低水準
- 物理的な農作業（剪定・機械操作・収穫）はAIの到達範囲外
- AIの効果は「情報処理・判断支援・パターン認識」に限定される
- 農業でのAI活用は「人を置き換える」のではなく「人の判断を助ける」方向が現実的

## 5. 総括: Optiens社への示唆

### 5.1 就労支援市場について

就労支援事業所向けの水耕栽培IoTシステム市場は存在するが、Optiens社の現在の事業方針（業務用ハーブ生産）とは異なるビジネスモデルである。将来的な事業展開の可能性として認識しておく程度でよい。

ただし、以下の点は参考になる:
- 訓練等給付費（月14-15万円/人）は、人件費を補助金で賄える仕組みとして強力
- B型事業所の開設はPhase 3以降の選択肢として検討の余地あり（ただし福祉事業の指定基準を満たす必要あり）

### 5.2 Tesla Optimusについて

2026年時点では完全に時期尚早。Optiens社のフェーズ・規模では投資対効果が合わない。2030年代に農業用途が実用化されても、小規模水耕栽培でのROIは疑問。ウォッチリストに留め、具体的なアクションは不要。

### 5.3 AI活用について

Optiens社が構築中のClaude Code x MCP x RPiの構成は、小規模水耕栽培のAI活用としては先進的。現実的な効果は環境モニタリング・養液管理・判断支援の自動化で全体の30-40%の省力化が見込める。収穫・梱包などの物理作業の自動化は当面不採算であり、Phase 2でのパート雇用が合理的。
