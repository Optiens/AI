# 用語マスタ（2026-05-17）

## 基本方針

公開ページ、メール、営業資料、管理画面では、サービス名と支払い単位を混同しない。

- 「スポット相談チケット」は支払い単位
- 「単発AI相談」「有償要件定義」「簡易実装・軽微な自動化」は提供メニュー
- 「AI診断官β」は入口、体験デモ、追加ヒアリング、診断メモ生成の仕組み
- 詳細版AI活用診断には60分MTGを含めない

## 禁止・注意表現

| 避ける表現 | 推奨表現 |
|---|---|
| AI診断官レビューを33,000円で販売 | AI診断官βは入口・体験デモ・詳細版の価値増強として提供 |
| 有償要件定義はチケット1枚 | 有償要件定義はチケット2枚 |
| 詳細版にMTGが付く | 詳細版はMTGなし。追加ヒアリングはAI診断官β、人間相談は単発AI相談 |
| 声を男性/女性に切り替えられる | 声の選択は現在できない |
| どんな環境でも音声で使える | 音声モードはβ版。静かな環境での利用を推奨 |
| 文字起こしを納品 | 会話内容を診断メモとして構造化 |
| 相談チケットは導入費へ充当 | 充当するのは有償要件定義の2枚分のみ |
