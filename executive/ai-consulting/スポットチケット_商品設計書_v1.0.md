# スポットチケット商品設計書 v1.0

## 1. 目的・ポジショニング

- 「30 万円未満のスモールニーズ」を受ける受け皿
- 業務相談 + 小規模作業（設定・調査・選定支援）を **単発で完結する** サービス
- 保守チケット（継続的な月額作業ベース）とは明確に別商品

### 保守チケットとの違い

| 項目 | 保守チケット | スポットチケット |
|---|---|---|
| 性質 | 月額契約・既存システム対象 | 単発購入・新規顧客含む |
| 契約形態 | 月額継続 | 単発購入（必要時のみ） |
| 1 件あたり単価 | ¥10,000〜¥15,000 | **¥30,000** |
| 想定利用シーン | 既存システムのメンテ・改修 | 「とりあえず相談したい」「進め方を聞きたい」+ 小作業 |
| 提供形式 | リモート作業＋報告 | オンライン MTG + 必要に応じた作業（Google Meet） |
| 「相談しかできない？」 | 作業ベース | **相談 + 小規模作業** を 1 パッケージ |

## 2. 商品ラインナップ

### 2.1 スポットチケット 1 枚

- **¥30,000 / 枚（税抜）**
- 1〜1.5 時間相当の対応（オンライン MTG + 必要なら小作業）
- 事前ヒアリングシート 1 枚（5 分で記入できるシンプル設計）
- 対応後の簡易議事録 1 枚（A4・PDF）

### 2.2 スポット 3 回パック

- **¥75,000 / 3 枚（税抜）**
- 単発単価より ¥15,000 お得（17% 引き）
- 有効期限 6 ヶ月（短期集中相談向け）
- 例：「導入前検討フェーズで月 1 回ずつ 3 ヶ月相談する」「3 つの業務を順番に見る」など

## 3. 想定利用シーン

- AI 活用診断（簡易版）の結果をもとに、具体的な進め方を相談したい
- 競合サービス（SaaS・代理店）の選定で第三者意見が欲しい
- 「Optiens に頼むほどではないが、専門家に相談したい」案件
- 30 万円未満の小規模ニーズ（LINE Bot 設置・Notion 整備・GAS スクリプト相談など）
- 業務フローの整理・棚卸し相談

## 4. オペレーション

### 4.1 申込→納品フロー

1. お客様が申込フォームから依頼（または既存顧客から直接メール）
2. 銀行振込で前金（¥30,000 または ¥75,000）受領
3. ヒアリングシート送付（A4 1 枚・Google Forms）
4. 日程調整 → オンライン MTG 実施（+ 必要に応じて小作業）
5. 対応後 3 営業日以内に簡易議事録送付
6. 適格請求書発行（インボイス対応）

### 4.2 工数想定

| 工程 | 時間 |
|---|---|
| ヒアリングシート確認・事前準備 | 20 分 |
| MTG | 60〜90 分 |
| 小作業（必要に応じて） | 〜30 分 |
| 簡易議事録作成 | 20 分 |
| **合計** | **2〜2.5 時間** |

時間単価換算：¥30,000 ÷ 2.5h = **¥12,000/h**（中堅エンジニア相場〜上限）

## 5. 紹介料の取扱い

**スポットチケットは紹介料の対象外**。

理由：
- 単価が低く（¥30,000）、紹介料を払うと粗利が圧迫
- 紹介者にとっても 1 件 ¥3,000〜¥6,000 の報酬は事務コストに見合わない
- ただし、スポット → 導入支援（30 万円以上）に発展した場合、その時点で紹介料規定が適用される

## 6. 紹介者への案内文例

> 「30 万円未満のお客様には、紹介料の対象外ですが、Optiens の **スポットチケット ¥30,000／枚（業務相談 + 小規模作業）** または 3 回パック ¥75,000 をご紹介いただけます。お客様にとっては手頃な相談メニュー、紹介者の方にとっては関係維持のきっかけになります。」
