# freee 事業所設定にインボイス番号を登録

**作成日**: 2026-05-11
**実施タイミング**: freee 会計を触るときに実施
**所要時間**: 約 5 分
**担当**: 代表

> ※ 本来は Google Tasks「経営管理」リストに登録予定だったが、
> Google Tasks MCP の OAuth クライアント削除エラー（`deleted_client`）で
> MCP 経由の登録に失敗。手動 Google Tasks 登録のバックアップとして本ファイルを作成。

---

## 手順

1. **freee 会計** にログイン（`accounting@optiens.com`）
2. **各種設定 → 事業所の設定**
3. **「適格請求書発行事業者番号」** 欄に以下を入力：

   ```
   T9090003003025
   ```

4. **保存**

## 関連メモ：Google Tasks MCP の問題

Google Tasks MCP（`C:/workspace/mcp-googletasks/build/index.js`）が
`deleted_client` エラーを返しました。

OAuth クライアントが Google Cloud Console 側で削除されている可能性があります。
別件で対応が必要：
- Google Cloud Console で OAuth 2.0 クライアント ID を再作成
- `~/.config/google-tasks-mcp/credentials.json` を更新
- MCP 再認証

これも別タスクとして管理が必要。
