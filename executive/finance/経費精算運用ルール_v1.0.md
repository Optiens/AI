# 経費精算運用ルール v1.0

**作成日**: 2026-05-13
**対象**: 合同会社Optiens 経費精算・記帳の標準ルール

---

## 基本原則

1. **法人費用と個人費用を明確に分離する**
2. **領収書は必ず freee に添付**（モバイル撮影 → アップロード）
3. **自宅兼事務所関連は事業使用割合 1/3 で按分**
4. **法人名義の口座・カードが第一選択。個人立替は次善**

---

## 1. 支払い経路別の処理パターン

### A. 法人口座（GMOあおぞら）からの直接支払い

| ケース | 処理 |
|---|---|
| 売上入金 | 自動仕訳（口座連携） |
| 振込・引き落とし | 自動仕訳 → 科目調整 |
| 通信費・サブスク等の定期引落 | 自動取込→科目振り分け（freee の自動推測を活用） |

仕訳パターン:
- 借方: 各経費科目 / 貸方: 普通預金（GMOあおぞら）

### B. freee カード Unlimited（法人カード）

| ケース | 処理 |
|---|---|
| AWS / Vercel / Supabase / OpenAI 等の月額 | カード明細から自動取込 |
| AliExpress 機材調達 | カード明細自動取込 |

仕訳パターン:
- 借方: 各経費科目 / 貸方: 未払金（freeeカード）

### C. 個人立替（最も慎重に運用）

**現状、Optiens の事業口座残高が少ないため多くがこのパターン。**

| ケース | 処理 |
|---|---|
| 個人クレカで法人費用を払った | 役員借入金で計上 |
| 現金で法人費用を払った | 役員借入金で計上 |
| 個人銀行口座から振込 | 役員借入金で計上 |

仕訳パターン:
- 借方: 各経費科目 / 貸方: 役員借入金（id: 1033228938）

**重要**: 領収書は必ず freee モバイルアプリで撮影してアップロード。`accounting@optiens.com` 経由でも可。

## 2. 按分対象経費（自宅兼事務所のため）

事業所が自宅と同一住所のため、以下は**事業使用割合 1/3** で按分する。

| 科目 | 按分対象 | 例 |
|---|---|---|
| 水道光熱費 | 電気代・ガス代・水道代 | 東京電力（毎月）→ 元額×1/3 |
| 通信費 | 自宅インターネット・固定電話 | NTT光（毎月）→ 元額×1/3 |
| 地代家賃 | （持家なら計上不可。賃貸なら按分） | 該当なし（持家） |

### 按分計算ルール
- 元額 × 1/3 → **小数点以下切り捨て** で経費計上
- 残り 2/3 は個人負担（仕訳に含めない）
- 仕訳の `description` に「自宅兼事務所・事業使用割合 1/3 按分」を必ず明記

### 按分根拠の保管
- 事業使用割合 1/3 の根拠（業務時間・使用面積等）を別途文書化（税務調査対応）
- 将来的に割合を変更する場合は、変更理由と適用開始日を記録

### 按分しない経費の例
- 業務専用機材（GPU・PC・カメラ等）: 全額経費
- 業務専用 SaaS / API（Claude / OpenAI 等）: 全額経費
- 業務専用回線（光回線2本目を業務専用化した場合）: 全額経費

## 4. 領収書管理ルール

### 保管必須
- **適格請求書発行事業者（インボイス）**からのもの: 7年間保管
- **3万円以上の取引**: 必ず適格請求書を受領（消費税仕入税額控除のため）

### freee への添付
- モバイルアプリ「freee 会計」で撮影 → 自動取込
- メール添付の場合: `accounting@optiens.com` に転送

### 紙の領収書
- 撮影してfreee に取込後、紙は段ボール箱に月別保管（来期決算後5年経過で廃棄可）

## 5. 役員借入金の運用

### 残高チェック
- 月次レビュー時に「役員借入金の月末残高」を必ず確認
- 増加し続けると将来の返済時に資金繰り負担

### 返済タイミング
- 法人の資金繰りに余裕ができたら、振込で代表に返済
- 返済時の仕訳: 役員借入金 / 普通預金

### 注意点
- 利息は原則つけない（利息相当額が代表の雑所得になるため面倒）
- 役員借入金は法人税法上、自己資本と同じ扱い（過小資本税制の対象は10億円超）

| タイミング | 作業 |
|---|---|
| 都度（経費発生時） | freee モバイルで領収書撮影＋仕訳起票 |
| 月末（25-31日） | 役員報酬の自動起票確認（既に12ヶ月分先付け済） |
| 月初（1-5日） | `/monthly-accounting-check` Skill 実行 |
| 月初（1-5日） | 月次レビュー（`月次レビューフォーマット_v1.0.md` 参照） |

---

## 7. NG行為

- 個人費用を法人で経費計上（脱税）
- 領収書なしで仕訳起票（証拠保全不可）
- 按分対象を全額計上（過大経費）
- 役員報酬を月額変更（法人税法上の定期同額給与違反 → 損金不算入）
- 同一取引の二重計上
